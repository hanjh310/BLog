OSX无共享的密钥情况下连接基于L2TP协议的VPN

发表于 2015-11-23 | 分类于 mac | |
字数统计: 98 | 阅读时长 ≈ 1

OSX需要配置的VPN（基于L2TP协议的IPSec），没有共享密钥的情况下，需要修改配置：
在/etc/ppp目录下新建一个文件options :

1：cd /etc/ppp
2：sudo vi options

写人如下内容：

plugin L2TP.ppp
l2tpnoipsec

保存后退出，最后把高级设置里面”通过VPN连接发送所有流量“钩上。

这时再点击连接就可以成功了
