IP命令集

发表于 2017-11-22 | 分类于 linux | |
字数统计: 1,232 | 阅读时长 ≈ 6

介绍

linux的ip命令和ifconfig类似，但前者功能更强大，并旨在取代后者。使用ip命令，只需一个命令，你就能很轻松地执行一些网络管理任务。ifconfig是net-tools中已被废弃使用的一个命令，许多年前就已经没有维护了。iproute2套件里提供了许多增强功能的命令，ip命令即是其中之一。

IP查询命令

  • addr 显示ip地址和属性信息（address的缩写）

  • link 管理和显示所有网络接口的状态

  • route 显示和修改路由表

  • maddr 管理和显示多播地址

  • neigh 显示邻居对象，也就是ipv4的arp表

  • help 显示子命令和字命令参数的帮助信息

多播地址管理

  • maddr add 添加静态链路层多播地址

  • maddr del 删除一个多播地址

修改地址和链接属性

  • addr add 添加一个地址

  • addr del 删除一个地址

  • link set 修改接口状态

调整和查看路由表

  • route add #添加一条路由表条目

  • route delete #删除一条路由条目

  • route replace #替换路由条目，如果没有定义就添加

  • route get #显示一个地址将要走的路由

管理arp表

  • neigh add #向arp表添加一条arp条目

  • neigh del #使arp条目失效

  • neigh replace #替换arp条目，如果不存在就添加

有用的命令（不一定来自iproute）

  • arping #向邻居发送arp请求

  • ethtool #查询或控制网络驱动程序和硬件设置

  • ss 显示套接字统计。以下选项可以组合使用

NETTOOLS与IPROUTE对比

NETTOOLS命令 IPROUTE命令
arp -a ip neigh
arp -v ip -s neigh

arp -i eth1 -d 192.168.1.1 ip neigh del 192.168.1.1 dev eth1
ifconfig -a ip addr
ifconfig eth0 down ip link set eth0 down

ifconfig eth0:0 192.168.1.2 ip addr add 192.168.1.2/24 dev eth0
netstat ss
netstat -neopa ss -neopa
netstat -g ip maddr
