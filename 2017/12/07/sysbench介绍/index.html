介绍

sysbench是一个基于LuaJIT的多线程基准测试脚本工具，常用于数据库的基准测试。但是也能用于其他非数据库的一些基准测试。目前版本1.0.10,包括：

  • oltp_*.lua: 数据库基准测试
  • fileio: 文件系统基准测试
  • cpu: cpu的基准测试
  • memory: 内存的基准测试
  • threads: 线程的调度测试

安装

  • Debian/Ubuntu

  • RHEL/CentOS

  • Fedora

  • macOS

用法

# sysbench
Python 抓取系统CPU负责和内存信息
